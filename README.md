# 🦷 DentBook - Dental Appointment Booking System

A modern web application to book and manage dental appointments easily.

---

## 🚀 Features

* 📅 Book dental appointments
* 👨‍⚕️ View available dentists
* 🛠️ Admin panel to manage appointments
* ✅ Approve / Cancel bookings
* 🔍 Search and filter appointments

---

## 🛠️ Tech Stack

* React + TypeScript
* Vite
* Tailwind CSS
* Supabase (Backend & Database)
* React Query

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/dental-booking.git
cd dental-booking
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run the Project

Start the development server:

```bash
npm run dev
```

Now open your browser and go to:

```
http://localhost:5173
```

---

## ⚙️ Environment Setup

Make sure to configure your Supabase credentials in:

```
.env
```

Example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```
